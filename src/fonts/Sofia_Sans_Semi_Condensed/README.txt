Sofia Sans Semi Condensed Variable Font
=======================================

This download contains Sofia Sans Semi Condensed as both variable fonts and static fonts.

Sofia Sans Semi Condensed is a variable font with this axis:
  wght

This means all the styles are contained in these files:
  SofiaSansSemiCondensed-VariableFont_wght.ttf
  SofiaSansSemiCondensed-Italic-VariableFont_wght.ttf
